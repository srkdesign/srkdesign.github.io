Wordpress -- это платформа для создания сайтов, где все управление контентом происходит через админ-панель. Wordpress облегчает пользовательский опыт и сокращает сроки разработки сайта.

## Для чего нужна локальная установка

- разработки собственных плагинов
- создания тем оформления
- поиска багов и ошибок на сайте (WP_DEBUG)
- безопасного тестирования, экспериментов с кодом и плагинами
- бекапов сайта с хостинга

## Варианты установки и их преимущества

Установить Wordpress локально можно с помощью сразу нескольких программ:

|                | Преимущества                                                                                                                                                                                                                                                                                                                                                                                   | Недостатки                                                                                                                                                          |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **LocalWP**    | <ul><li>легкая установка</li> <li>понятный интерфейс</li> <li>не нужно настраивать базы данных и phpMyAdmin вручную</li> <li>автоматически подписывает SSL сертификаты</li> <li>управление сразу несколькими сайтами</li> <li>есть возможность клонирования и экспорта сайтов</li> <li>хорошо подходит для клиентских сайтов и обучения</li> <li>поддерживает Windows, MacOS и Linux</li></ul> | <ul><li>ограниченная настройка</li> <li>требует больше ресурсов</li></ul>                                                                                           |
| **Wampserver** | <ul><li>можно полностью подстроить сервер под текущий проект</li></ul>                                                                                                                                                                                                                                                                                                                         | <ul><li>предлагает скачать сразу несколько программ во время установки</li> <li>требует настройки баз данных и phpMyAdmin</li></ul>                                 |
| **Openserver** | <ul><li>легкая установка</li> <li>контроль над всеми компонентами сервера</li> <li>полностью поддерживает MariaDB</li> <li>несколько версий PHP/MySQL</li> <li>виртуальные хосты</li> <li>полностью бесплатный</li></ul>                                                                                                                                                                       | <ul><li>настройка требует больше времени чем LocalWP</li> <li>интерфейс может показаться сложным</li> <li>обновления компонентов могут приходить не сразу</li></ul> |
| **MAMP**       | <ul><li>понятный интерфейс</li> <li>поддерживает Windows и MacOS</li></ul>                                                                                                                                                                                                                                                                                                                     | <ul><li>для расширенных функций нужна платная версия</li> <li>требует настройки баз данных и phpMyAdmin</li></ul>                                                   |

Поскольку LocalWP поддерживает все платформы дальше речь пойдет именно о нем.

## Установка Local WP

1. Перейдите на сайт [localwp.com](https://localwp.com)
2. Нажмите на кнопку "Download For Free" и заполните данные во всплавающем окне
3. После заполнения данных нажмите на кнопку "Get it now"
4. Дождитесь загрузки установщика и следуйте инструкциям после его запуска

## Создание сайта LocalWP

1. Запустите LocalWP
2. Нажмите на кнопку с плюсом в левом нижнем углу экрана

<ImageCaption
  image={local_2}
  format={format}
  quality={quality}
  caption="Нажмите на кнопку + в левом нижнем углу"
/>

3. Если хотите создать сайт с 0, выберите "Create new site" (Создать новый сайт) и нажмите "Continue" (Продолжить)
4. _(Опционально) Если хотите запустить локально бекап сайта, выберите ZIP файл и перетащите его в прямоугольник с обводкой_

<ImageCaption
  image={local_3}
  format={format}
  quality={quality}
  caption="Выберите какой сайт хотите создать: с нуля или из .zip файла"
/>

5. **Настройка сайта.** Выберите название для сайта, из которого автоматически будет создан домен для вашего сайта (который можно ввести и вручную). Выберите путь к вашему сайта на диске (может быть абсолютно любая папка, по умолчанию это Local Sites)

<ImageCaption
  image={local_4}
  format={format}
  quality={quality}
  caption="Название сайта обязательно нужно указывать латинскими буквами"
/>

6. **Настройка среды.** В разделе "Custom" вы можете настроить PHP версию, web сервер (Apache или Nginix) и базу данных. Если вам это не важно, можете оставить версии по умолчанию, выбрав "Preferred"

<ImageCaption
  image={local_5}
  format={format}
  quality={quality}
  caption="Настройка среда Wordpress во время создания сайта с LocalWP. Если не знаете, что это, лучше оставить знаечния по умолчанию"
/>

7. **Настройка Wordpress.** В этом разделе вам будет предложено ввести логин и пароль администратора Wordpress, выбрать язык сайта

<ImageCaption
  image={local_6}
  format={format}
  quality={quality}
  caption="Настройка Wordpress во время создания сайта с LocalWP"
