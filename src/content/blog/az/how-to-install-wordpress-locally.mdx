WordPress — sayt yaratmaq üçün məşhur platformadır. Bütün kontent admin paneli vasitəsilə idarə olunur. WordPress istifadəçi üçün sadədir və saytın hazırlanma müddətini azaldır.

## Niyə lokal quraşdırma lazımdır?

- öz plaginlərinizi yaratmaq
- tema və dizaynlar yaratmaq
- saytda xətaları tapmaq və düzəltmək (WP_DEBUG)
- kod və plaginləri təhlükəsiz sınaqdan keçirmək
- saytın hostinqdəki backup-ını çıxarmaq

## LocalWP ilə WordPress-in yüklənilməsi

1. [localwp.com](https://localwp.com) saytına daxil olun
2. "Download For Free" düyməsini klikləyin və açılan pəncərədə məlumatları doldurun
3. Məlumatlarınızı doldurduqdan sonra "Get it now" düyməsini klikləyin
4. Quraşdırıcı yüklənəndən sonra işə salın və təlimatlara əməl edin

## LocalWP-də yeni saytının yaradılması

1. LocalWP-ni işə salın
2. Sol alt küncdəki + (plyus) düyməsini klikləyin

<ImageCaption
  image={local_2}
  format={format}
  quality={quality}
  caption="Sol alt küncdəki + (plyus) düyməsini klikləyin"
/>

3. Saytı 0-dan yaratmaq istəyirsinizsə, “Create new site” (Yeni sayt yarat) seçin və daha sonra “Continue” (Davam et) düyməsini klikləyin
4. _(İstəyə görə) ZIP fayldan backup-ını işə salmaq istəyirsinizsə, faylı sahəyə sürükləyin_

<ImageCaption
  image={local_3}
  format={format}
  quality={quality}
  caption="Hansını yaratmaq istədiyinizi seçin: sıfırdan sayt və ya .zip fayldan"
/>

5. Saytın adını daxil edin. Bu ad avtomatik olaraq saytın domeninə çevriləcək (istəsəniz özü də yaza bilərsiniz). Sonra, saytın kompüterdə yerləşəcəyi qovluğu seçin (standart olaraq Local Sites)

<ImageCaption
  image={local_4}
  format={format}
  quality={quality}
  caption="Saytın adını mütləq latın hərfləri ilə daxil edin"
/>

6. **Mühitin quraşdırması.** “Custom” bölməsində PHP versiyasını, web serveri (Apache və ya Nginx) və verilənlər bazasını seçə bilərsiniz. Əgər sizə vacib deyilsə, “Preferred” seçimi ilə standart parametrləri saxlaya bilərsiniz.

<ImageCaption
  image={local_5}
  format={format}
  quality={quality}
  caption="LocalWP ilə sayt yaradarkən WordPress mühitinin quraşdırılması. Nə olduğunu bilmirsinizsə, standart parametrləri saxlamaq daha yaxşıdır"
/>

7. **WordPress quraşdırması.** Admin istifadəçi adı və şifrənizi daxil edin və saytın dilini seçə bilərsiniz.

<ImageCaption
  image={local_6}
