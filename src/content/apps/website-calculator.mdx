Удобный инструмент для веб-дизайнеров и разработчиков, который позволяет быстро рассчитать стоимость проекта и оформить результат в презентабельной таблице для отправки клиенту.

На данный момент поддерживается только Windows. Версия для macOS требует дополнительной настройки для корректной работы.

Программа с открытым исходным кодом — можете проверить ее безопасность самостоятельно.

## Преимущества

- **Экономия времени** — расчет занимает всего пару минут
- **Презентабельный файл** — аккуратная таблица с детализацией стоимости по страницам и блокам
- **Автоматический перевод** — финальный файл формируется на русском (₽) или английском ($), в зависимости от выбранной валюты
- **Темная и светлая темы оформления**

## Возможности калькулятора

- Расчет стоимости по количеству страниц и блоков
- Учет уровня сложности проекта по шкале от 0 до 5
- Поддержка двух валют: рубли и доллары
- Обновление стоимости всего проекта по мере добавления блоков в режиме реального времени
- Экспорт результата в виде Excel-файла (. xlsx)
- Финальный файл включает структуру проекта и детальную стоимость по каждому элементу

## Кому подходит

- Фрилансерам, которым важно быстро и точно оценивать проекты
- Упрощения общения с клиентами
